## 1. Vision

La mission de ThermoPilot AI est de transformer la rénovation énergétique des bâtiments grâce à la data et l'intelligence artificielle.

Le secteur du bâtiment représente près de **44 %** de la consommation énergétique en France.
Malgré l'urgence climatique, les audits énergétiques restent longs, coûteux et peu accessibles.

Notre vision est de créer une plateforme capable de transformer les données d'un bâtiment en stratégie énergétique en quelques minutes.

ThermoPilot AI devient ainsi le **copilote énergétique** des copropriétés et des professionnels du bâtiment.

---

## 2. Mission

La mission de l'entreprise est de :

- automatiser les audits énergétiques
- faciliter la prise de décision pour les rénovations
- réduire les coûts d'analyse technique
- accélérer la transition énergétique

Nous voulons permettre à chaque acteur du bâtiment de prendre les bonnes décisions énergétiques en quelques clics.

---

## 3. Problème

Aujourd'hui, les audits énergétiques présentent plusieurs limites :

### Processus long

Un audit complet peut prendre **2 à 4 semaines**.

### Dépendance aux experts

Les calculs nécessitent des ingénieurs spécialisés.

### Données dispersées

Les informations sont souvent réparties entre :

- factures
- plans
- relevés techniques
- diagnostics

### Décision difficile

Les copropriétés ont du mal à choisir entre différents scénarios de rénovation.

---

## 4. Solution

ThermoPilot AI est une **plateforme SaaS d'analyse énergétique automatisée**.

Elle permet de :

- analyser les caractéristiques d'un bâtiment
- calculer sa performance énergétique
- simuler différents scénarios de rénovation
- générer automatiquement des rapports techniques

Les utilisateurs obtiennent en quelques minutes :

- classe énergétique
- scénarios de travaux
- estimation des coûts
- gains énergétiques
- retour sur investissement

## 8. Produit

La plateforme ThermoPilot AI comporte plusieurs modules.

### Module analyse bâtiment

Permet de saisir les caractéristiques du bâtiment :

- surface
- année de construction
- type de chauffage
- isolation
- menuiseries

Le système calcule automatiquement la consommation énergétique estimée.

---

### Module simulation travaux

Permet de tester plusieurs scénarios :

- isolation thermique par l'extérieur
- remplacement chaudière
- installation pompe à chaleur
- remplacement menuiseries

Chaque scénario affiche :

- gain énergétique
- coût estimé
- retour sur investissement
- nouvelle classe énergétique

---

### Module génération de rapport

La plateforme génère automatiquement :

- rapports d'audit énergétique
- synthèse pour assemblée générale
- scénarios comparatifs
- documents PDF exportables

---

### Module dashboard

Les utilisateurs disposent d'un tableau de bord pour :

- gérer leur portefeuille de bâtiments
- suivre les audits réalisés
- comparer plusieurs scénarios

## 10. Technologie

La plateforme repose sur une architecture cloud moderne.

### Frontend

| Technologie | Rôle |
|-------------|------|
| Next.js | Framework React SSR |
| React | UI composants |
| TypeScript | Typage statique |
| Tailwind CSS | Styles utilitaires |

Avantages : performance élevée, interface moderne, maintenance facilitée.

### Backend

Basé sur Python avec :

- **FastAPI** — API REST performante
- **SQLAlchemy** — ORM et gestion BDD
- **Celery** — Tâches asynchrones

Avantages : rapidité de développement, excellente gestion des calculs scientifiques, compatibilité avec l'écosystème IA.

### Base de données

**PostgreSQL** est utilisé pour :

- gestion des utilisateurs
- stockage des audits
- scénarios énergétiques

Base robuste, scalable, avec support natif JSONB et UUID.

### Intelligence artificielle

Les calculs et prédictions utilisent :

- **Pandas** — manipulation de données
- **NumPy** — calculs numériques
- **Scikit-learn** — modèles prédictifs

Ces bibliothèques permettent d'analyser les données énergétiques et d'estimer les gains de rénovation.

### Infrastructure cloud

L'infrastructure repose sur **AWS** :

| Service | Usage |
|---------|-------|
| EC2 | Serveurs applicatifs |
| S3 | Stockage de fichiers |
| RDS | Base de données managée |
| CloudFront | Distribution CDN |

Cette architecture permet une forte scalabilité.
